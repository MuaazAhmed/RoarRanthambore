# Static Images Folder

## Structure:
- /static/home.jpg         → Hero background image for the homepage
- /static/gallery/         → Place gallery images here (e.g. tiger1.jpg, tiger2.jpg, etc.)
- /static/logos/           → Logo files (e.g. logo.png)
- /static/pages/           → Background images for other pages (e.g. booking.jpg, about.jpg)

## Gallery image naming convention:
gallery1.jpg, gallery2.jpg, gallery3.jpg ... gallery8.jpg
